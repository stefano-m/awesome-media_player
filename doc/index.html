Create a new MediaPlayer object that implements the org.mpris.MediaPlayer2.Player DBus interface.

media_player.MediaPlayer:metadata ()
Return the metadata associated to the current track.

Returns:

    A table containing the metadata.
media_player.MediaPlayer:info ()
Return useful information about the track.

Returns:

    A table with the following string attributes (if available in the track's metadata):

    • album: name of the album
    • title: title of the song
    • year: song year
    • artists: comma-separated list of artists (may be just one artist)
    • length: total lenght of the track as HH:MM:SS
